## ✨ 主要功能

- **🔍 实时进程监控**  
  基于 `psutil` 高性能检测，只要进程名包含关键词即视为运行中，精准识别闪退。
- **🔄 自动重启机制**  
  检测到进程消失后，自动清理残留进程并重新启动应用程序。
- **🖱️ 自动化初始化流程**  
  启动后自动执行预设的鼠标点击序列（登录 -> 切换列表 -> 全选 -> 成品），无需人工干预。
- **⚙️ 可视化配置界面**  
  基于 `CustomTkinter` 打造的现代化深色模式 UI，支持一键获取屏幕坐标、测试进程及路径选择。
- **💾 配置持久化**  
  所有设置（路径、坐标、间隔时间）自动保存至同级目录的 `.ini` 配置文件，下次启动无需重复设置。
- **📝 实时运行日志**  
  内置日志窗口，实时显示监控状态、点击操作及异常信息，方便排查问题。

## 🚀 快速开始

### 1. 克隆或下载项目
将 `WenJingRestart.py` 下载到本地任意文件夹。

### 2. 首次运行
双击脚本或在终端运行：
```bash
py WenJingRestart.py
```

### 3. 配置指南
程序启动后，请按以下步骤完成配置：

1.  **设置目标程序**：
    *   **进程名关键词**：输入文镜进程名的字符（例如 `wenjing` 或 `main`，无需 `.exe`）。点击“测试检测”确认能否找到进程。（填入程序完整路径会自动填入进程名关键词，如自动填写有误请手动配置）
    *   **程序完整路径**：点击“浏览...”选择文镜的 `.exe` 启动文件。
2.  **获取操作坐标**（关键步骤）：
    *   将文镜软件打开并停留在**默认初始界面**。
    *   在助手界面中，依次点击各步骤（登录、切换列表等）旁的 **📍 获取当前鼠标坐标** 按钮。
    *   *提示：点击按钮后有 3 秒倒计时，请将鼠标移动到软件界面对应的按钮位置等待自动记录。*
3.  **调整设置**（可选）：
    *   **监控间隔**：默认为 60 秒，即每隔多久检查一次进程状态。
    *   **启动等待**：软件启动后等待多少秒再开始执行点击操作（视电脑性能调整，默认 15 秒）。

### 4. 开始守护
配置完成后，点击绿色按钮 **▶ 开始守护**。
*   助手可最小化或留在后台运行。
*   当日志显示 `✅ 监控已启动` 时，即可放心离开。
*   若需停止，点击 **⏹ 停止** 。

## ⚠️ 重要注意事项

1.  **坐标依赖性**：本工具使用**绝对坐标**进行点击。如果文镜软件更新导致界面布局变化，或者您改变了软件窗口的位置/分辨率，**务必重新获取坐标**，否则点击可能失效。
2.  **屏幕分辨率**：建议在固定的屏幕分辨率下使用，避免因分辨率变化导致坐标偏移。
3.  **权限问题**：如果文镜软件以管理员身份运行，本助手可能也需要以**管理员身份运行**才能正确检测和杀掉进程。
4.  **防干扰**：在执行自动点击流程时（刚重启后的几十秒内），请勿手动移动鼠标或操作键盘，以免干扰自动化脚本。
